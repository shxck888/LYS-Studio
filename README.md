# LYS Studio 麗的舞蹈 官方網站

汐止 LYS Studio 舞蹈教室形象網站。靜態網頁（HTML / CSS / JavaScript），無需建置工具。

## 頁面

- `index.html`：首頁（形象主視覺、特色、課程預覽、競賽榮耀、作品集）
- `courses.html`：課程介紹
- `schedule.html`：每週課表
- `teachers.html`：師資介紹
- `gallery.html`：作品集與競賽成績
- `contact.html`：聯絡與報名（地址、地圖、LINE / IG / FB）

## 本機預覽

直接用瀏覽器開啟 `index.html` 即可，或啟動簡易伺服器：

```bash
python3 -m http.server 8000
# 瀏覽 http://localhost:8000
```

## 部署到 GitHub Pages

1. 將本資料夾推送到 GitHub repo。
2. 進入 repo 的 **Settings → Pages**。
3. Source 選 **Deploy from a branch**，分支選 `main`、資料夾選 `/ (root)`，儲存。
4. 稍候幾分鐘，網站會發布在 `https://<你的帳號>.github.io/<repo 名稱>/`。
